## Мета роботи
Головною метою роботи є опанування базового інструментарію Data Science мовою Python для автоматизованого збору, очищення (Data Cleaning) та комплексного аналізу великих масивів даних. Робота спрямована на практичне вивчення можливостей бібліотеки `pandas` та середовища `Jupyter Notebook` на прикладі роботи з метеорологічними даними NOAA та великим датасетом споживання електроенергії.

## Структура проекту та Data Pipeline
Роботу розділено на два логічні блоки, кожен з яких реалізовано в окремому інтерактивному зошиті:

### Частина 1: Аналіз вегетаційного індексу (Part1.ipynb)
- **Автоматизований парсинг (urllib):** Скрипт автоматично завантажує 27 CSV-файлів з сервером NOAA (індекс VHI з 1981 по 2024 рік). Реалізовано механізм перевірки наявності файлів для запобігання дублюванню мережевих запитів та додавання timestamp до назв.
- **Очищення та реіндексація:** Сирі дані парсяться у DataFrame, очищаються від залишкових HTML-тегів, позбавляються NaN значень та порожніх рядків. Індексація областей динамічно переводиться з англійського стандарту NOAA у формат української абетки (1 - Вінницька область). Дані по Києву та Севастополю відфільтровуються.
- **Аналітичні вибірки:** Реалізовано функції для отримання часових рядів VHI за конкретний рік та область, формування вибірок за діапазоном років, а також розрахунку базової статистики (мінімум, максимум, середнє, медіана).

### Частина 2: Аналіз споживання електроенергії (Part2.ipynb)
- **Обробка великих даних:** Завантаження та розпакування архіву `Individual Household Electric Power Consumption` (~20 МБ). Конвертація розрізнених колонок Date та Time у єдиний формат `datetime`.
- **Комплексні фільтрації:** Реалізовано складні логічні вибірки (наприклад, фільтрація за часовими проміжками, пороговими значеннями потужності та порівняння споживання між різними групами приладів). Для складної багатоступеневої вибірки застосовано зрізи (slicing) для вибору кожного n-го елемента.
- **Профілювання продуктивності:** Для кожної функції вибірки інтегровано оцінку часу виконання за допомогою магічної команди `%timeit`.
- **Машинне навчання (Препроцесинг):** - Проведено нормалізацію (MinMaxScaler) та стандартизацію (StandardScaler) числових ознак датасету математичним шляхом.
  - Розраховано коефіцієнти кореляції Пірсона та Спірмена для аналізу лінійної та монотонної залежності між ознаками.
  - Виконано перетворення категоріального атрибута (день тижня) у числовий формат за допомогою One-Hot Encoding (`pd.get_dummies`).

## Вимоги до середовища
- **ОС:** Windows / Linux / macOS
- **Інтерпретатор:** Python 3.10+
- **Залежності:** Всі необхідні бібліотеки вказані у файлі `requirements.txt` (зокрема `pandas`, `scipy`, `urllib`).

Інструкція із запуску

1\. Створити віртуальне середовище: python -m venv venv

2\. Активувати середовище: venv\\Scripts\\activate

3\. Встановити залежності: pip install -r requirements.txt

4\. Запустити Jupyter: jupyter notebook
