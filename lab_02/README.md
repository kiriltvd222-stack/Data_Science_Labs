Лабораторна робота №2: Аналіз даних (NOAA та Power Consumption)



Завдання

Отримання та очищення даних (VHI індекс та споживання електроенергії). Формування вибірок, розрахунок статистики, стандартизація даних та One Hot Encoding.



Вимоги до системи
- ОС: Windows / Linux / macOS

\- Версія інтерпретатора: Python 3.10+

\- Менеджер пакетів: pip



Інструкція із запуску

1\. Створити віртуальне середовище: python -m venv venv

2\. Активувати середовище: venv\\Scripts\\activate

3\. Встановити залежності: pip install -r requirements.txt

4\. Запустити Jupyter: jupyter notebook
