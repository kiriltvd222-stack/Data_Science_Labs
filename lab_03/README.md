# Лабораторна робота №3. Візуалізація та аналіз багатовимірних даних
---

## Мета роботи
Головною метою даної роботи є опанування практичних навичок проведення експлоративного аналізу даних (EDA). Основна увага приділена методам очищення вибірки (Data Cleaning) та побудові складних візуалізацій для виявлення прихованих закономірностей у багатовимірних структурах даних за допомогою бібліотек Matplotlib та Seaborn.

## Опис досліджуваного датасету
Для аналізу було обрано Automobile Data Set з репозиторію UCI Machine Learning. Даний набір даних є репрезентативним прикладом багатовимірної структури, оскільки містить 26 атрибутів різних типів:
- Категоріальні ознаки: тип кузова (body-style), тип пального (fuel-type) тощо.
- Числові атрибути (цілочисельні та дійсні): потужність (horsepower), об'єм двигуна (engine-size), вартість (price).
Особливістю датасету є наявність пропущених значень, що вимагало проведення етапу попередньої підготовки.

## Етап 1: Очищення та підготовка даних (Data Cleaning)
Перед початком візуалізації було реалізовано алгоритм очищення, що включав наступні кроки:
1. Ідентифікація пропусків: У вихідному файлі пропуски були представлені символом "?". Всі такі входження були конвертовані у формат np.nan для коректної обробки бібліотекою Pandas.
2. Корекція типів даних: Стовпці з ціною, потужністю та іншими числовими показниками, які зчиталися як текстові об'єкти, були примусово переведені у формат float або int.
3. Стратегія імпутації: Для збереження обсягу вибірки пропуски в числових колонках були заповнені середнім арифметичним (mean), а в категоріальних — модою (mode). Це забезпечило відсутність Null-значень без викривлення статистичного розподілу.

## Етап 2: Аналітична візуалізація
У ході роботи було розроблено шість типів графіків, що дозволили провести всебічний аналіз:

1. Діаграма розсіяння (Scatter Plot): Візуалізація залежності ціни від об'єму двигуна. Результати демонструють чітку позитивну лінійну кореляцію, що підтверджує прямий вплив об'єму двигуна на вартість авто.
2. Лінійний графік (Line Plot): Аналіз тенденції ціни відносно кінських сил. На відсортованих даних спостерігається експоненціальне зростання вартості, особливо в сегменті понад 150 к.с.
3. Гістограма розподілу потужності: Дані були розбиті на 5 специфічних діапазонів. Аналіз показав, що ринок орієнтований на середньопотужний сегмент (80–120 к.с.), який є найбільш масовим.
4. Багатовимірна візуалізація (4D Scatter): Складний графік, що поєднує об'єм двигуна (X), витрату пального (Y), колір за типом кузова та розмір точки за ціною. Це дозволило наочно побачити низьку паливну ефективність дорогих автомобілів з великими двигунами.
5. Матриця взаємозв'язків (Pairplot): Використано для сегментації характеристик за типом пального. Графік дозволяє швидко оцінити кластеризацію дизельних та бензинових моделей.
6. Теплова карта (Heatmap): Побудована на основі кореляції Пірсона. Вона математично підтвердила сильний зворотний зв'язок між ціною та показниками економії пального (city-mpg).
